# 릴리즈 노트

## 2026-06-12

### 새로운 기능

- 지도 우상단 목록 버튼으로 **필지 목록**을 열어 전체 필지를 한 화면에서 훑어볼 수 있습니다.
- 목록에서 **지번·이름·그룹명 검색**, **색상 필터**, **지번순·색상순·면적순 정렬**로 원하는 필지를 빠르게 찾을 수 있습니다.
- 목록의 행을 탭하면 **필지 시트(그룹 소속이면 그룹 시트)가 목록 위에 바로 열리고**, 시트를 닫으면 검색·필터 상태가 그대로 남습니다.
- 같은 지번이 여러 필지로 존재하면 **식별 번호를 병기**해 행을 구분합니다.
- 목록의 면적 컬럼은 **㎡/평/a/ha 단위 전환**을 지원합니다. 면적 값은 추후 공공 데이터 연동 후 채워지며, 그 전까지는 "-"로 표시됩니다.
- 필지 여러 개를 탭으로 골라 **그룹**으로 묶을 수 있습니다. 그룹에는 **이름·메모·색**을 정할 수 있고, 지도에서 한 덩어리로 표시됩니다.
- 그룹에 속한 필지를 탭하면 **그룹 시트**가 열려 이름·메모·색·표시 방식과 **합계 면적**을 한꺼번에 보고 수정할 수 있습니다.
- 그룹을 만든 뒤에도 **필지 추가** 모드로 필지를 더하거나, 멤버 목록에서 필지를 뺄 수 있습니다.
- 필요 없어진 그룹은 **그룹 해체**로 풀 수 있으며, 각 필지의 개별 색칠·이름·메모는 그대로 남습니다.
- 그룹을 만들다 **취소**하면 지도가 만들기 전 상태로 정확히 되돌아갑니다.
- 지도에서 필지를 탭하면 **필지 시트**가 열려 **색칠·이름·메모를 직접 편집**하고 저장할 수 있습니다. v2의 첫 편집 기능으로, 저장한 내용은 모두의 화면에 실시간 반영됩니다.
- 색을 칠한 필지는 **채움/테두리** 두 가지 표시 방식 중에서 고를 수 있습니다.
- 집·농기계·수도 같은 중요한 필지는 **고정**하고 5개 분류 40종 **아이콘**을 붙여 한눈에 알아볼 수 있습니다. 고정 필지는 앞으로 제공될 초기화 기능에서 색상·이름·아이콘이 보호됩니다.
- 시트에서 필지의 **면적과 지목**을 확인하고, 면적 단위를 **㎡/평/a/ha**로 전환할 수 있습니다. 선택한 단위는 기기에 저장되어 유지됩니다.
- 넓은 화면(태블릿·PC)에서는 시트가 오른쪽 패널로 열리며, **시트를 연 채 지도 이동·확대·다른 필지 선택**이 가능합니다.

## 2026-06-11

### 새로운 기능

- **지적편집도 v2** 파일럿 버전을 처음 공개합니다.
- 앱을 열면 서버에 저장된 **필지 색칠과 그룹**이 지도에 그대로 표시됩니다. 이번 버전은 **보기 전용**이며, 색칠·그룹 편집 화면은 다음 업데이트에서 제공됩니다.
- 다른 사람이 변경한 **필지 색칠·메모·그룹·팔레트**가 새로고침 없이 **실시간으로 내 화면에 반영**됩니다.
- 내가 방금 저장한 변경이 서버에서 되돌아와도 화면이 깜빡이거나 입력 중인 내용을 덮어쓰지 않습니다.
- 다른 기기에서 내가 보고 있던 작업공간을 닫으면, 남아 있는 첫 작업공간으로 **자동 전환**됩니다.
- 지도 화면 우상단 버튼으로 **릴리즈 노트**를 열어 버전별 변경사항을 확인할 수 있습니다.
- 보구곶리 필지 지도를 빠르게 불러와 표시합니다.
- 지도를 **드래그로 이동**하고, **마우스 휠·두 손가락 핀치·＋/− 버튼**으로 확대/축소할 수 있습니다.
- 필지를 **탭(클릭)**하면 선택되어 강조 표시되고, 빈 곳을 탭하면 선택이 해제됩니다.
- 지도에 필지 **지번**이 표시됩니다. 지도를 확대할수록 작은 필지의 지번도 점차 나타납니다.
- 지번 글자에 **흰색 테두리**를 둘러 색칠된 필지 위에서도 또렷하게 읽힙니다.

### 개선

- 지도 배경색을 기존 서비스와 동일한 **따뜻한 미색 톤**으로 맞췄습니다.
- 필지 색칠·그룹 표시가 기존 서비스와 동일하게 보이도록 지도 그리기를 안정화했습니다.
- 기존 서비스에서 **지도 오른쪽 영역의 필지가 탭으로 선택되지 않던 문제**를 수정했습니다.
- 초기 데이터를 불러오는 동안에는 지도 탭이 동작하지 않도록 해, 로드 중 어긋난 조작이 발생하지 않습니다.
- 기존 서비스에서 **내가 바꾼 팔레트 색이 내 화면에 한 번 더 적용되어 깜빡이던 동작**을 고쳐, 자기 변경은 다시 적용하지 않습니다.

### 내부 개선

- 여러 **작업공간(탭)**을 지원하기 위한 새로운 데이터 저장 구조를 준비했습니다.
- 닫은 탭을 나중에 복원할 수 있는 **히스토리** 저장 기반을 마련했습니다.
- 개발 환경과 실제 서비스가 같은 서버 코드를 사용하도록 내부 구조를 정리해, 환경에 따라 동작이 달라지는 문제를 원천 차단했습니다.
- 지도 그리기 엔진을 새로 작성하면서 그룹 외곽선 계산을 캐싱해, 화면 이동·확대 시 불필요한 재계산을 없앴습니다.
- 지번 글자의 줄바꿈 측정과 그룹 덩어리 계산도 캐싱해, 라벨이 많아도 지도 이동이 느려지지 않도록 했습니다.
- 터치·마우스·펜 입력을 하나의 처리 경로로 통합해, 기기에 따라 지도 조작이 달라지거나 중복 반응하던 여지를 없앴습니다.
- 화면 상태 관리를 중앙 스토어로 통합해, 앞으로의 실시간 동기화와 편집 화면 개발의 기반을 마련했습니다.
- 필지 데이터 정리 규칙을 화면과 서버가 같은 코드로 공유해, 저장 시 일부 정보가 사라질 수 있던 경로를 차단했습니다.
- 실시간 연결의 상태(연결 중·연결됨·오류)를 내부에서 추적해, 앞으로 연결 상태 표시와 자동 복구 안내를 붙일 수 있는 기반을 마련했습니다.
