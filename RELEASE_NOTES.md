# 릴리즈 노트

## 2026-06-11

### 새로운 기능

- **지적편집도 v2** 파일럿 버전을 처음 공개합니다.
- 앱을 열면 서버에 저장된 **필지 색칠과 그룹**이 지도에 그대로 표시됩니다. 이번 버전은 **보기 전용**이며, 색칠·그룹 편집 화면은 다음 업데이트에서 제공됩니다.
- 지도 화면 우상단 버튼으로 **릴리즈 노트**를 열어 버전별 변경사항을 확인할 수 있습니다.
- 보구곶리 필지 지도를 빠르게 불러와 표시합니다.
- 지도를 **드래그로 이동**하고, **마우스 휠·두 손가락 핀치·＋/− 버튼**으로 확대/축소할 수 있습니다.
- 필지를 **탭(클릭)**하면 선택되어 강조 표시되고, 빈 곳을 탭하면 선택이 해제됩니다.
- 지도에 필지 **지번**이 표시됩니다. 지도를 확대할수록 작은 필지의 지번도 점차 나타납니다.
- 지번 글자에 **흰색 테두리**를 둘러 색칠된 필지 위에서도 또렷하게 읽힙니다.

### 개선

- 지도 배경색을 기존 서비스와 동일한 **따뜻한 미색 톤**으로 맞췄습니다.
- 필지 색칠·그룹 표시가 기존 서비스와 동일하게 보이도록 지도 그리기를 안정화했습니다.
- 기존 서비스에서 **지도 오른쪽 영역의 필지가 탭으로 선택되지 않던 문제**를 수정했습니다.
- 초기 데이터를 불러오는 동안에는 지도 탭이 동작하지 않도록 해, 로드 중 어긋난 조작이 발생하지 않습니다.

### 내부 개선

- 여러 **작업공간(탭)**을 지원하기 위한 새로운 데이터 저장 구조를 준비했습니다.
- 닫은 탭을 나중에 복원할 수 있는 **히스토리** 저장 기반을 마련했습니다.
- 개발 환경과 실제 서비스가 같은 서버 코드를 사용하도록 내부 구조를 정리해, 환경에 따라 동작이 달라지는 문제를 원천 차단했습니다.
- 지도 그리기 엔진을 새로 작성하면서 그룹 외곽선 계산을 캐싱해, 화면 이동·확대 시 불필요한 재계산을 없앴습니다.
- 지번 글자의 줄바꿈 측정과 그룹 덩어리 계산도 캐싱해, 라벨이 많아도 지도 이동이 느려지지 않도록 했습니다.
- 터치·마우스·펜 입력을 하나의 처리 경로로 통합해, 기기에 따라 지도 조작이 달라지거나 중복 반응하던 여지를 없앴습니다.
- 화면 상태 관리를 중앙 스토어로 통합해, 앞으로의 실시간 동기화와 편집 화면 개발의 기반을 마련했습니다.
- 필지 데이터 정리 규칙을 화면과 서버가 같은 코드로 공유해, 저장 시 일부 정보가 사라질 수 있던 경로를 차단했습니다.
